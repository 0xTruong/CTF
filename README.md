 Dùng vòng for để kiểm tra từ 3100-3200
 
 Gán localhost cho key SERVER ( localhost là máy chủ hiện tại )
 
 PORT=$i gắn từng giá trị từ 3100-3200
 
 (echo  > /dev/tcp/$SERVER/$PORT) >& /dev/null && là mở kết nối tcp đến cổng $PORT trên máy chủ localhost, Nếu cổng mở lệnh này thực thi thành công, nếu thất bại lỗi sẽ chuyển hướng sang  /dev/null để ẩn thông báo

 echo "Port $PORT open": In ra các cổng mở.
